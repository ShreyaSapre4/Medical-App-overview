Overview

I developed a comprehensive medical appointment booking website using the following technologies:

  • Frontend: ReactJS
  • Backend: NodeJS, ExpressJS
  • Database: MongoDB
  • Authentication: JWT, Bcrypt
  • Styling: SCSS
  • HTTP Requests: Axios

Patient Side

The patient side of the website provides a user-friendly platform with the following features:

  • User authentication to ensure secure access.
  • Display of doctors from various departments.
  • A form to fill out for booking appointments.
  • Easy navigation to find and schedule appointments with doctors.

Admin Side

The admin side includes robust features for efficient management:

  • Secure authentication for admin access.
  • A comprehensive dashboard displaying a list of appointments.
  • Functionality to approve or reject appointments.
  • Management of doctor listings, including adding or deleting doctors from different departments.
